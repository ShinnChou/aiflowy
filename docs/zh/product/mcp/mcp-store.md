## 二、安装MCP
点击单个MCP卡片，进入MCP详情页，点击右上角的 **安装到系统** 按钮，会提示**添加成功**
![mcp_store_4.png](resource/mcp_store_4.png)
此时我们在这里添加的mcp就成功添加到我们的菜单MCP中去了,然后我们就可以启动服务
